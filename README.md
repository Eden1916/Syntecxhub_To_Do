A working to-do project using html, css & js. The to-do-list app saves the data to local storage so the data won't be deleted or missed after refresh. And the tick button is used to mark the task as completed so the task will be blur when it is completed and the cross button will appeare when the tick button is clicked that is used to make the task un completed or undo to un completed task. The pencil buttno is used to edit the task. And the basket button is used to cancel the task.
